# Etapa 1.1 — Modalidade ESPIRITUAL

**Cor canônica:** dourado (#854F0B / c-amber 800 — peso, não brilho)
**Conceito:** Forjar a alma. Propósito, conexão, silêncio, ritual.

---

## Sub-categorias (4)

1. **Propósito** — meditação ativa, intenção, valores, contemplação
2. **Conexão** — gratidão, vínculo, presença com outros e com o todo
3. **Silêncio** — quietude, solitude consciente, vazio escolhido
4. **Ritual** — práticas repetidas com sentido, gestos cerimoniais

## REGRAS DE COMBINAÇÃO

Quando o sistema gera 1 missão Espiritual:

1. Sorteia **sub-categoria**
   - Pesos sugeridos: propósito 30% / conexão 25% / silêncio 25% / ritual 20%
2. Sorteia **3 sub-tarefas dentro da sub-categoria**
3. Aplica escala por rank (sub-tarefas com escala 0 são puladas)
4. Sorteia **1 título** do pool da sub-categoria
5. Sorteia **1 quote** do pool do pilar Espiritual

Exemplo:
- Sub-categoria: **Silêncio**
- Sub-tarefas: silencio_total + sentar_imovel + escutar_corpo
- Rank D: 20min silêncio + 10min imóvel + 10min body scan
- Título: **"A Cela do Sol"**
- Quote: *"O Vazio teme silêncio que escolhe ser silêncio."*
